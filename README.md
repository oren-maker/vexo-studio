# VEXO STUDIO

פרויקט Next.js 14 עם TypeScript, Tailwind ו-Supabase.

## התקנה

```bash
npm install
cp .env.example .env.local
npm run dev
```

## העלאה ל-Vercel

1. צור ריפו ב-GitHub והעלה את הקוד.
2. ב-Vercel: New Project → Import מ-GitHub.
3. הוסף את המשתנים מ-`.env.example` בהגדרות Environment Variables.
4. Deploy.
